# MATH


## Factorial
- n! = 1 * 2 * 3 * (n-1) * n
- 수학적으로 0!과 1!의 값은 1


## GCD (Greatest Common Di)

### 유클리드 호제법
- 두 개의 자연수에 대한 최대공약수를 구하는 대표적인 알고리즘
- 두 자연수 A, B에 대하여 (A>B) A를 B로 나눈 나머지를 R
- 이때 A와 B의 최대공약수는 B와 R의 최대공약수와 동일
- 유클리드 호제법의 아이디어를 그대로 재귀 함수로 작성 가능
    - 예시 : GCD(192,162)
    
        | Step | A | B |
        | :----: | :-: | :-: |
        | 1 | 192 | 162 |
        | 2 | 162 | 30 |
        | 3 | 30 | 12 |
        | 4 | 12 | 6 |
